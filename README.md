# CodeAlpha - Hangman Web Experience

A premium, responsive, web-based Hangman game built using a Flask backend API and a glassmorphic dark-themed frontend.

[![Deploy to Render](https://render.com/images/deploy-to-render.svg)](https://render.com/deploy?repo=https://github.com/venu1920/CodeAlpha_Hangman-Game)

---

## Features

- **Categorized Word Bank**: Words are dynamically drawn from multiple categories (Programming, Animals, Countries, and Web Dev).
- **Session-Based State Management**: Secure server-side game validation prevents cheating.
- **Score Tracker**: Automatically tracks wins and losses across rounds.
- **Modern Premium Design**: Visually stunning dark mode with glowing accents and smooth micro-animations.
- **Responsive Layout**: Designed to adapt seamlessly to desktop, tablet, and mobile screens.

## Local Setup

### Prerequisites
Make sure you have Python installed.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/venu1920/CodeAlpha_Hangman-Game.git
   cd CodeAlpha_Hangman-Game
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the development server:
   ```bash
   python app.py
   ```

4. Open your browser and navigate to `http://127.0.0.1:5000` to play.
